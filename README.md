
# Personal Portfolio

This is a modern, responsive personal portfolio website built with React and Vite.

## Features

- **Premium Design**: Glassmorphism, smooth animations, and a cohesive color palette.
- **Responsive Navigation**: Mobile-friendly menu with smooth scrolling.
- **Hero Section**: Eyecatching introduction with 3D-like elements.
- **Projects Showcase**: Interactive grid with hover effects.
- **Skills Overview**: Categorized tech stack display.
- **Contact Form**: Functional-looking contact section.

## Setup

1. **Install dependencies**:
   ```bash
   npm install
   ```

2. **Run development server**:
   ```bash
   npm run dev
   ```

3. **Build for production**:
   ```bash
   npm run build
   ```

## Customization

- **Colors & Theme**: Edit `src/index.css` variables (`:root`).
- **Content**: Update data in `src/components/*.jsx` files.
- **Images**: Replace unsplash URLs with your own images.
